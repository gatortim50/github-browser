### Github-Browser
- User may login
- View their personal repos
- Search for repos
- View list of repos
- Logout

# rename env file to .env
- edit the .env file and fill in the appropriate values below
- USERNAME=XXX  // your email or github user name
- PASSWORD=XXX  // your password
